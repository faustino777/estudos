Principais formatações

Negrito / Destaque

Nesta frase temos um termo em negrito usando a tag "B" (não semântica).

Nesta frase temos um termo em destaque usando a tag "strong" (semântica)

Itálico / Ênfase

Nesta frase temos um termo em itálico usando a tag "I" (não semântica)

Nesta frase temos um termo em ênfase usando a tag "em" (semântica)

Texto marcado

Podemos criar também um texto marcado usando a tag "mark".

E em outro parágrafo também.

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno neste parágrafo.

Texto deletado

Podemos marcar um texto como excluído para indicar que ele dever sser lido, mas não considerado, com a tag "del".

Texto inserido

Podemos marcar um texto como inserido parar dar uma ênfase e undicar que ele foi adicionado depois, com a tag "ins" (semântica).

Existe também o sublinhado com a tag "U" (não semântica).

Texto sobrescrito

Podemos escrever coisas sobrescritas com a tag "sup".
Ex: x20+3

Texto subscrito

Podemos escrever também coisas subescritas com a tag "sub".
Ex: H20
